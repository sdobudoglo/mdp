1. Система поиска товара:
Обширный и простой поиск, который доступен каждому пользователю/гостю сайта. Поиск работает и выдает результаты по названию, цене, категории.
[пользователь может легко найти конкретно тот товар который его интересует исходя из тех фильтров которые представлены. В строку поиска пишется название товара(можно не полное) и в результат выводится все что совпало с тем что написал пользователь. Если такого товара не существует выводится соответствующее окно с ошибкой «Данного товара нет в нашем магазине»]
2. Фильтрация товара:
Пользователю доступны фильтры для более удобного поиска товара. 
Фильтры: цена, название, категория, дата добавления, бренд, характеристики)
[При поиске товара пользователь может настроить фильтр для более подходящего выбора который ему необходим. Фильтр настраивается в подходящем поле справа. Доступна фильтрация по цене, производителю, бренду, дате добавления, некоторые характеристики]
3. Корзина:
У пользователя есть возможность добавлять интересующий  или удалять товар из своей корзины.
[Корзина служит для суммирования цены и количества выбранного товара, так же из нее можно оформить заказ, добавить количество уже выбранного товара, добавить промокод, удалить ненужный товар. Вывод итоговой цены с учетом промокода или скидки на товар]
4. Оформление заказа:
После выбора товара и добавления в корзину необходимо оформить заказ. В оформление заказа входит ввод персональных данных(ФИО, город, номер телефона), выбор места и способа доставки товара.
[Для зарегистрированных пользователей:
Пользователю необходимо ввести свои данные для входа в учетную запись(номер телефона пароль) это необходимо для удобства чтобы каждый раз не заполнять все необходимые данные для заказа
Для новых пользователей:
Пользователю у которого нет учетной записи на сайте необходимо ввести все данные необходимые для заказа вручную(Имя, фамилия, отчество, номер телефона, e-mail, выбрать способ доставки(самовывоз или доставка по адресу) при доставке по адресу необходимо ввести данные(Город, улица, номер дома, квартира) доставка на почту( город, индекс почты))
Далее для обоих пользователей(зарегистрированных и новых) необходимо выбрать способ оплаты(наличные, перевод на карту) 
Так же есть возможность оставить какой либо комментарий к заказу
После этого необходимо подтвердить заказ нажав на соответствующую кнопку]
5. Оповещение о заказе:
После заказа выбранного товара пользователь получит на почту письмо с информацией о заказе.
[На почту пользователю придёт письмо с оповещением о том что заказ успешно создан в котором будет описан товар который заказал пользователь(название, количество и цена в сумме)]
6. Личный кабинет пользователя:
Пользователь может редактировать свои данные в личном кабинете который доступен после регистрации. ФИО пользователя, город, номер телефона, аватар на сайте.
[После регистрации пользователю станет доступен личный кабинет в котором он может указать информацию о себе, это необходимо для удобства при оформлении заказа(чтобы каждый раз не вводить данные снова) Так же в личном кабинете находится баланс бонусов за покупки который совершал пользователь на сайте. Редактирование главной информации для входа в аккаунт(номер телефона, пароль)]
7. Обратная связь с тех-поддержкой.
Пользователь может отправить письмо на почту тех поддержки с интересующим вопросом.
[При возникновении вопроса или какой либо проблемы пользователь может отправить письмо на адрес тех поддержки (texpod@mail.ru) и в определенный промежуток времени ему придет ответ с решением его проблемы]
8. Каталог товара. Пользователь может просматривать разделы с интересующим товаром.
[на главной странице доступен каталог товаров. Он служит для разделения всех имеющихся товаров на категории, что необходимо для удобства при поиске конкретного товара из конкретной категории. Меню доступно слева на главной странице сайта. При выборе какой либо категории высвечиваются только те товары которые ей принадлежат]
9. Обширное и понятное описание товара.
К каждому товару приводится его полное описание и фото. Каждый пользователь может его просматривать.
[При выборе конкретного товара пользователем выводится его полное описание, характеристики и фото. Необходимо для более подробного описания товара. Так же на странице конкретного товара можно оставить комментарий и оценку товара]

10. Система скидок, промокодов.
[На сайте есть возможность получать промокод в честь какого то значимого события(день рождение пользователя) промокод состоит из определенной фразы которую можно ввести в корзине и получить бонус(скидка или подарок)
Система скидок предназначена для более быстрой распродажи товаров которых много на складе информация о скидках доступна на главном экране где выводятся хиты продаж и товары со скидками. Длительность скидок ограниченная по времени]
11. Доставка.
Выбор способа доставки.
[При оформлении заказа пользователю необходимо выбрать способ доставки его покупки, для новых пользователь информацию нужно каждый раз вводить снова, для зарегистрированных пользователей возможно использование шаблона который редактируется в личном кабинете
Есть 3 способа доставки(самовывоз(пользователю необходимо лично приехать на склад и забрать свой товар), доставка по адресу(пользователь заполняет все необходимые поля и курьер доставляет товар ему домой), доставка на почту (необходимо указать город и индекс почты и дождаться поступления товара в отделение почты, после этого забрать его]
12. Информативный функционал:
- Рассылка новостей пользователям
[при регистрации есть возможность поставить галочку для получение информации о поступлении новых товаров, скидках, промокодах и тд на почту пользователя]
- Публикация новостей на сайте
[Предусмотрен специальный раздел где публикуются все новости о магазине, товарах и каких то интересных событиях(распродажи, акции, конкурсы]
